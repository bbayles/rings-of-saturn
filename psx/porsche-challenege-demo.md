This is a patch for the *Euro Demo 14* disc that modifies its
demo of *Porsche Challenge*. It's not normally playable, but this patch allows you
to take control.

The demo files are dated 1996-10-08, several months older than the final version's (1997-07-16).
A title card calls the demo 50% complete.

The Releases page on this repository has an XDelta patch that can be applied with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `Euro Demo 14 (Europe) (Track 1).bin`
* __XDelta patch__: `porsche-challenge-demo.xdelta`

After starting the demo disc, select *Porsche Challenge* from the main menu to play.
