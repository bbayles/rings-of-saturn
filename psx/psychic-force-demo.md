The releases page of this repository has a patch for *Psychic Force Tentou Demo-ban*, a demo disc version of *Psychic Force*.

The patch starts up in "Debug game" mode, which allows you to choose any character or stage. Normally the demo restricts which characters and stages you can play.

Apply the patch with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `Psychic Force (Japan) (Demo).bin`
* __XDelta patch__: `psychic-force-demo.xdelta`

The build is from 1996-07-12, 40 days before the final version (1996-08-21).
