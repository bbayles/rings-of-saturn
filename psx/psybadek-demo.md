This is a patch for the PAL demo of *Psybadek* (SLED-01488). 
Normally this demo limits which levels and menus you can access, but this patch
remove the restrictions and allows exploration of the entire build.

The demo's files are from is from 1998-08-06, about a month before the final version.

Several elements don't match up with the final version:
* The skate park has warps to every world. In the final game, these are in a separate area.
* The level entrances for some of the worlds have different layouts.
* Only one type of projectile can be encountered.
* The Continue screen accepts different passwords.

The Releases page on this repository has an XDelta patch that can be applied with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `Psybadek (Europe) (En,Fr,De,Es,It) (Demo) (Track 1).bin`
* __XDelta patch__: `psybadek-demo.xdelta`
