This is a patch for the *Codemasters Demo Disk* that removes the restrictions on its
*Micro Maniacs* preview.

The demo is based on a "75% complete" version of the game. Without the menu
restrictions, you can see a number of unfinished elements.
See [the writeup](https://32bits.substack.com/p/ffb1602a-0bb4-43de-b86c-1dbb24402d58)
for details.

The Releases page on this repository has an XDelta patch that can be applied with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `Codemasters Demo Disk (Europe) (Track 1).bin`
* __XDelta patch__: `micro-maniacs-demo.xdelta`

After booting the demo disc, select *Micro Maniacs* from the main menu.
