This is a patch for the *DemoDemo PlayStation Vol. 13* demo disc that extends
the preview of *Horned Owl*.

Normally this preview allows you to play part of a single stage, but this patch enables
the menus and allows you to select any of the missions.

The demo build is from 1995-10-04, 26 days earlier than the final Japanese build.
There are several unfinished elements:
* Several menuitems have different labels.
* The Training Room mode doesn't work.
* You can access the Gun Adjust screen even if you don’t have a light gun plugged in.
* Character and map overlays don't appear during gameplay.

The Releases page on this repository has an XDelta patch that can be applied with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `DemoDemo PlayStation Vol. 13 (Japan) (Track 1).bin`
* __XDelta patch__: `horned-own-demo.xdelta`

After booting the demo disc:
* Press Start to access the main menu.
* Press Start to go to the Playable Demos screen.
* Press Down to highlight the Horned Owl demo.
* Press Start twice to launch it.
