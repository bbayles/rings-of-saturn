The Release page for this repository contains patches for two PlayStation versions of *Ms. Pac-Man: Maze Madness*.
* The NTSC-U PlayStation version
* The NTSC-U PlayStation demo

The patches enables a hidden level select menu. Access it through the pause menu during gameplay.
Some levels seem to be unused, and others have unfinished or corrupt textures.

Apply the PlayStation patches with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5).

For the NTSC-U retail version:
* __Original file__: `Ms. Pac-Man Maze Madness (USA) (Track 1).bin`
* __XDelta patch__: `maze-madness-psx.xdelta`

For the NTSC-U demo version:
* __Original file__: `Ms. Pac-Man Maze Madness (USA) (Demo) (Track 1).bin`
* __XDelta patch__: `maze-madness-psxdemo.xdelta`

See screenshots and get technical details about the patch on [my site](https://32bits.substack.com/p/8158c3de-c825-449f-8a5a-4fc6ce29a12b).
