This is a patch for the *DemoDemo PlayStation Vol. 16* demo disc that extends
the preview of *Motor Toon Grand Prix 2*.

Normally this preview allows you to race in one mode (Free Run) as
one character (Captain Rock) on one course (Toon Village). This patch unlocks:
* Single Race, Time Attack, and Two Players Battle modes.
* Seven more characters.
* An extra course (Toon Island II).

The demo build is from 1996-01-01, more than three months before the
final one (1996-04-25). There are a number of visible differences:
* The menu items in this build don't match the final build.
* Some of the characters have different names.
* The character controls and camera angles have not been finalized.

The Releases page on this repository has an XDelta patch that can be applied with  [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `DemoDemo PlayStation Vol. 16 (Japan) (Track 1).bin`
* __XDelta patch__: `motor-toon-2-demo.xdelta`

After booting the demo disc:
* Press Start to access the main menu.
* Press Start to go to the Playable Demos screen.
* Press Down to highlight the Motor Toon Grand Prix 2 demo.
* Press Start twice to launch it.
