This is a patch for *Official U.S. PlayStation Magazine Demo Disc 13* that extends
the preview of *Test Drive 5*.

Normally this preview limits the modes, tracks, and vehicles you can access.
This patch removes the demo restrictions and allows for the build to be fully explored.
* All cars can be selected.
* Three tracks can be selected.
* All menu items are enabled.

The demo's files are from is from 1997-07-13, 2 months before the final U.S. version.

There are several differences with the retail version. See [the writeup](https://32bits.substack.com/p/eb4323b7-fe5a-45c8-8f80-b986ef167318) for details.

The Releases page on this repository has an XDelta patch that can be applied with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `Official U.S. PlayStation Magazine Demo Disc 13 (USA) (Track 1).bin`
* __XDelta patch__: `test-drive-5-demo.xdelta`

After booting the demo disc, select *Test Drive 5* from the list of demos on the main menu.
