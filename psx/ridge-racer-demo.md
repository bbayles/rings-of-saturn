This is a patch for the *DemoDemo PlayStation Vol. 1* preview CD that modifies its
demo of *Ridge Racer* to remove its restrictions. The patch enables access to the menu
screens, which allow you to change courses, cars, and adjust configuration.

The demo disc contains a more-or-less full build of the *Ridge Racer* that predates
the final one by three weeks. There are lots of differences and unfinished elements.

According to the CD filesystem timestamps, this build is from 1994-10-19, making it
some of the earliest available PlayStation software.

The Releases page on this repository has an XDelta patch that can be applied with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `DemoDemo PlayStation Soukan-gou Vol. 1 (Japan) (Track 1).bin`
* __XDelta patch__: `ridge-racer-proto.xdelta`

A writeup with more details is available at [my website](https://bbayles.com/32bits).
