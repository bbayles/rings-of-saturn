The releases page for this repository has a patch for the *Toys "R" Us: "Test Drive" Promotion* demo disc.
It lifts the demo restrictions for the version of *Steel Reign* and makes more of it
playable.

The demo is based on a build from at least a few months before the final version.
It has several differences, including:
* An extra difficulty mode
* A different level listed in the multiplayer Arena Selection screen
* A different name for the "Anaconda" tank

The patch also enables the hidden "Secrets" menu, which enables debug features.

Apply the patch with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `Toys R Us Test Drive Promotion (USA) (Track 1).bin`
* __XDelta patch__: `steel-reign-demo.xdelta`

Select __Steel Reign__ from the demo menu at startup to play.

See screenshots and get technical details about the patch on [my site](https://32bits.substack.com/p/401d0ac8-b3c3-4c1e-b2d8-5ef76acc4d6a).
