The releases page for this repository contains a patch for the preview version of *Tales of Eternia* for PlayStation
that enables debug features and allows for exploration beyond the normal demo limits.

Apply it with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `Tales of Eternia (Japan) (Demo) (Track 1).bin`
* __XDelta patch__: `toe-demo.xdelta`

A writeup with more details is available at [Rings of Saturn](https://32bits.substack.com/p/a9f98a16-09cf-4452-bece-cf87298bd12e).
