This is a patch for the "Purple" PlayStation Picks demo disc. It makes the rolling demo of *Destruction Derby* playable.

The demo is based on a prerelease build of the game from 1995-07-23, which seems to be the earliest available build.

The Releases page on this repository has an XDelta patch that can be applied with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `PlayStation Picks (USA) (SCUS-94952) (Track 01).bin`
* __XDelta patch__: `dd-demo-01.xdelta`

See [my blog](https://32bits.substack.com) for details about how the patch works.
