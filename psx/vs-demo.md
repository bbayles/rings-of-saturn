This is a patch for *Official U.S. PlayStation Magazine Demo Disc 03* that extends
the preview of *Vs*.

Normally this preview limits the modes, characters, and stages you can access.
This patch removes the demo restrictions and allows for the build to be fully explored.

The demo's files are from is from 1997-08-06, 12 weeks before the final U.S. version.

There are several unfinished elements:
* There's an extra song in the Options menu's "Music Select" list.
* Several characters are listed as having different heights.
* Some menu items have different labels.
* The Defensive Fire Indicator is missing during gameplay.

The Releases page on this repository has an XDelta patch that can be applied with [Delta Patcher](https://github.com/marco-calautti/DeltaPatcher/releases/tag/v3.1.5):
* __Original file__: `Official U.S. PlayStation Magazine Demo Disc 03 (USA) (Track 1).bin`
* __XDelta patch__: `vs-demo.xdelta`

After booting the demo disc, select *Vs* from the list of demos on the main menu.
