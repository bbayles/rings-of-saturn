The releases page for this repository contains a patch for *Ms. Pac-Man: Maze Madness*.
It enables a hidden level select menu. Access it through the pause menu during gameplay.
Some levels seem to be unused, and others have unfinished or corrupt textures.

Apply the patch with [Universal Dreamcast Patcher](https://github.com/DerekPascarella/UniversalDreamcastPatcher/releases/tag/1.8):
* __Original file__: `Ms. Pac-Man - Maze Madness v1.000 (2000)(Namco)(US)[!].gdi`
* __XDelta patch__: `maze-madness-dc.dcp`

See screenshots and get technical details about the patch on [my site](https://32bits.substack.com/p/8158c3de-c825-449f-8a5a-4fc6ce29a12b).
